# Street

## Getting started

#### 1. Install dependencies:

With `yarn` (quicker)

```
yarn install
```

or with `npm` (slower):

```
npm install
```

#### 2. Run Gulp for development:

```
yarn start
```

or

```
npm start
```

#### 3. Open development URL - [`http://localhost:9000/`](http://localhost:9000/).

## Scripts

#### Running a project for development:

```
npm start
```

#### Build a project for production:

```
npm run build
```

#### Creating of a production-ready zip-archive `build.zip`:

```
npm run zip
```
